 true Is background visible or not

 Background align
Properties
Name Type Attributes Default Description
x string <optional>
 "left" Horizontal align. "left", "right", "center"
y string <optional>
 "bottom" Vertical align. "top", "bottom", "middle"
pos object <optional>
 Background position
Properties
Name Type Attributes Default Description
x number <optional>
 0 X coordinate in axis units of background position
y number <optional>
 0 Y coordinate in axis units of background position
fit number <optional>
 cl.Background.FIT_ALL Background fit type. Can be:
cl.Background.FIT_NONE = 0; no fit, image size not changed
cl.Background.FIT_STRETCH = 1; stretch image to fit chart size
cl.Background.FIT_ALL = 2; image will be scaled to fit chart size using aspect ratio
cl.Background.FIT_FILL = 2; image will be scaled to fill all chart using aspect ratio
Properties:
